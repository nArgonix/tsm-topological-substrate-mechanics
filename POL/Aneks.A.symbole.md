# Aneks A: Zintegrowany spis symboli i oznaczeń

Poniższa tabela zestawia wszystkie fundamentalne pojęcia, zmienne i stałe materiałowe wykorzystywane w formalizmie Topologicznej Geometrodynamiki 0-Matrycy (TGM).

| Symbol | Opis | Wymiar / Uwagi (Odwołanie w tekście) |
| --- | --- | --- |
| $\mathcal{M}^4$ | 4-wymiarowa rozmaitość różniczkowa, kontinuum 0-Matrycy | Euklidesowa w Stanie Zero, Riemannowska lokalnie (Rozdz. 1.1) |
| $x^1, x^2, x^3$ | Współrzędne przestrzenne naszej izotropowej 3-brany | Metry |
| $x^4$ | Współrzędna czwartego wymiaru przestrzennego (kierunek ortogonalny) | Metry |
| $g_{ab}$ | Metryka Riemanna na $\mathcal{M}^4$ o sygnaturze dodatniej (++++) | Bezwymiarowa (Rozdz. 1.1) |
| $\sigma_{ab}$ | Całkowity symetryczny tensor naprężenia Matrycy | J/m³ (Rozdz. 1.1) |
| $\Sigma$ | Ślad tensora naprężenia ($\sigma^a_{\,a}$), makroskopowe ciśnienie tła | Skalar, J/m³ (Rozdz. 1.1, 10.2) |
| $\sigma_{44}$ | Ortogonalny naciąg w osi czwartego wymiaru (źródło grawitacji) | J/m³ (Rozdz. 5.1) |
| $\sigma_{\text{max}}$ | Maksymalne naprężenie plastyczne (próg rozerwania węzłów) | Rzędu gęstości Plancka; warunkuje plazmoidy (Rozdz. 9.2) |
| $K_{abcd}$ | Tensor modułów sprężystości 4-wymiarowego ośrodka | J/m³ (Rozdz. 2.1) |
| $\alpha(\mathbf{x})$ | Lokalna funkcja sprężystości 0-Matrycy (ewolucyjny analog stałej G) | Zmienna w czasie kosmicznym (Rozdz. 5.1) |
| $\Phi$ | Potencjał grawitacyjny w izotropowej 3-branie | J/kg (Rozdz. 5.1) |
| $\phi_a$ / $\psi$ | Pole przesunięcia (odkształcenia) osnowy / pole osnowy solitonu | Generuje transformację fazy splotu (Rozdz. 2.1, 2.7) |
| $c_0$ | Prędkość światła (fal poprzecznych) w zrelaksowanym Stanie Zero | Definicja stałej: 299 792 458 m/s (Rozdz. 1.7) |
| $c_{\text{lok}}$ | Lokalna prędkość propagacji fal zależna od naprężeń tła | m/s (Rozdz. 1.2, 4.5) |
| $\gamma$ | Współczynnik kontrakcji przestrzennej (odpowiednik Lorentza) | Bezwymiarowy opór falowy Matrycy w 3D (Rozdz. 1.4) |
| $\epsilon_0, \mu_0$ | Przenikalność elektryczna i magnetyczna w zrelaksowanym Stanie Zero | Stałe konwencjonalne (Rozdz. 1.7) |
| $\epsilon_{\text{eff}}$ | Efektywna przenikalność osnowy naciągniętej | $\epsilon_{\text{eff}} = \epsilon_0 (1 + \kappa B^2 + \dots)$ (Rozdz. 1.5, 4.5) |
| $\kappa$ | Stała sprzężenia magneto-elastycznego (podatność skrętna) | T⁻²; zakłada się $\kappa > 0$ (Rozdz. 1.5, 4.5) |
| $B$ | Indukcja pola magnetycznego (wirowość odkształceń Matrycy) | T (tesla) (Rozdz. 4.2) |
| $f_0$ | Częstotliwość wzorca drgań w Stanie Zero | Hz (Rozdz. 1.6) |
| $f_{\text{lok}}$ | Lokalna częstotliwość oscylacji (definiuje czas emergentny) | Hz (Rozdz. 1.6) |
| $\zeta$ | Stała sprzężenia naprężenie–częstotliwość | Wymiar odwrotności gęstości energii (Rozdz. 1.6) |
| $\mathcal{W}$ | Liczba splotu (niezmiennik topologiczny, chiralność) | Liczba całkowita; dla elektronu $ |
| $H$ | Niezmiennik Hopfa dla pola wektorowego splotu | Bezwymiarowy; dla elektronu $H = 1$ (Rozdz. 2.7, Aneks D) |
| $\chi$ | Podstawowa stała topologiczna Matrycy (energia bazowego splotu) | Wymiar J; ściśle determinuje masę elektronu (Rozdz. 1.7, 2.2) |
| $\mathcal{P}(\mathbf{x})$ | Topologiczna gęstość wykluczenia przestrzennego | Wykładnicza bariera przeciw nałożeniu węzłów (Rozdz. 3.2) |
| $g$ | Stała topologicznego odpychania (dla zakazu Pauliego) | Bezwymiarowa stała geometryczna (Rozdz. 2.4, 3.2) |
| $q, e$ | Ładunek elektryczny / ładunek elementarny | Całkowity strumień wektora skrętu fazowego wokół węzła (Rozdz. 4.1) |
| $\sigma$ | Napięcie struny topologicznej (szyjki pęcherzyka kawitacyjnego) | J/m; ok. 1 GeV/fm w oddziaływaniach silnych (Rozdz. 6.3) |
| $V_{ij}$ | Nieliniowa uwięziona energia interferencji naprężeń gromad | Wymiar J; wyjaśnia zjawisko ciemnej materii (Rozdz. 8.3) |
| $z, z_{\text{src}}, z_{\text{IGM}}, z_{\text{obs}}$ | Przesunięcie ku czerwieni (całkowite oraz składowe naprężeniowe) | Bezwymiarowe (Rozdz. 10.3) |
| $H_0$ | Obserwowana stała Hubble'a | Zdefiniowana jako współczynnik tempa relaksacji Matrycy (Rozdz. 10.2) |
| $Q$ | Dobroć rezonansu wnęki elektromagnetycznej | Bezwymiarowa (Rozdz. 11.1) |
| $S^{\alpha\beta}$ | Tensor sferyczności rozpadu | Analiza topologii zdarzeń w akceleratorach (Rozdz. 6.12) |
| $\lambda_1, \lambda_2, \lambda_3$ | Wartości własne tensora sferyczności | Bezwymiarowe; $\lambda_1 \ge \lambda_2 \ge \lambda_3$ (Rozdz. 6.12) |
| $S, A$ | Sferyczność i planarność | Bezwymiarowe (Rozdz. 6.12) |
